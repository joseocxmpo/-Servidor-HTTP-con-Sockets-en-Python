Acerca del Servidor

Implementación

Este servidor HTTP está implementado usando el módulo socketserver de Python con ThreadingMixIn para manejar múltiples conexiones concurrentes usando hilos.

Características

  • Soporte para método GET
  • Detección automática de tipos MIME
  • Manejo de errores HTTP (400, 403, 404, 405, 500)
  • Headers HTTP conformes a RFC 9110
  • Protección contra directory traversal
